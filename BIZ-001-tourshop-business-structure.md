## 4. 견적·예약 플로우

### B2C / B2B 공통 플로우
```
Step 1. 여행 종류 선택
        🎿 스키·스노보드
        ⛳ 골프
        🥾 트래킹·등반
        🏄 어드벤처·액티비티
        ✈️ 자유 여행 (맞춤)

Step 2. 나라 선택
        (여행 종류별 운영 나라 목록 제시)

Step 3. 세부 항목 단계별 선택
        호텔·스키장·골프장·교통·가이드·
        리프트티켓·액티비티·보험·입장료

Step 4. 출발일·기간·인원 입력

Step 5. 금액 자동 산출
        원가 + 마진율 자동 적용

Step 6. 공유
        고객: 1인 단가·총액
        담당자: 원가+마진 전체 내역
```

### 내부 직원 플로우
```
동일 플로우 → 원가만 산출
→ 담당자가 최종 마진 추가 후 고객 전달
```

## 5. 마진율 구조

### 적용 기준
- **항목 + 나라 조합별** 마진율 개별 설정
- 위치: `wiki/costs/markup-guide.md`

### 채널별 마진 차등
| 채널 | 마진 적용 |
|------|---------|
| B2C | 항목+나라 조합 마진율 100% 적용 |
| B2B | B2C 마진율에서 % 할인 |
| 내부 | 마진 미적용 (원가만) |

## 6. 원가 구성 항목

```
여행 총 원가
├── ✈️ 항공권 (airticket-rates)
├── 🏨 호텔 숙박 (hotel-rates)
├── 🚌 현지 교통 (transport-rates)
├── 👤 가이드·인솔자 (guide-fees)
├── 🎿 리프트티켓·스키패스 (lift_skipass-rates)
├── ⛳ 골프 그린피·카트피·캐디피 (golf-rates)
├── 🏄 액티비티 (activity-rates)
├── 🛡️ 보험 (insurance-rates)
└── 🎫 관광지 입장료 (entrance-fees)
```

## 8. 향후 서비스 로드맵

### 1단계 (현재)
- Cowork 기반 내부 에이전트 운영
- 원가 DB 구축 시작
- 수동 견적 → 반자동 전환

### 2단계
- 앱·웹 B2C 서비스 오픈
- B2B 파트너 포털 구축
- 자동 견적 시스템 연동

### 3단계
- AI 챗봇 고객 응대 자동화
- 여행안내서 완전 자동화
- MCP 서버 기반 외부 서비스 연동
