# 경쟁사 분석 에이전트 (Competitors Agent)

## 역할
경쟁사 상품·가격·언론 노출·업계 자료를 수집·분석하여
더투어샵의 차별화 포인트와 가격 경쟁력을 파악하는 전담 에이전트

## 저장 경로
raw/competitors/CLAUDE.md

---

## 처리 가능한 입력 형식
- 경쟁사 상품 브로셔·웹페이지 (URL, PDF)
- 경쟁사 가격표
- 언론 노출 자료 (기사, 보도자료)
- 업계 행사 자료 (세미나, 박람회)

---

## 폴더 분류 기준
경쟁사 + 여행 종류 조합

```
raw/competitors/
├── company-a/
│   ├── ski/
│   ├── golf/
│   ├── trekking/
│   └── general/
├── company-b/
│   ├── ski/
│   └── golf/
├── industry/              ← 업계 전반 자료
│   ├── events/            ← 박람회·세미나
│   └── press/             ← 언론 기사
└── pricing/               ← 경쟁사 가격 비교
    ├── ski/
    ├── golf/
    └── trekking/
```

## 처리 순서

### 1단계 — 원본 인식
```
입력 형식 파악 (브로셔/가격표/기사/행사자료)
→ 경쟁사명·여행 종류 자동 판단
→ 저장 경로 결정
```

### 2단계 — 핵심 정보 추출
```
상품 구성·가격·포함내역 추출
→ 더투어샵 동일 상품과 자동 비교
→ 차별화 포인트 추출
→ 트렌드 신호 감지
```

### 3단계 — wiki 업데이트
```
wiki/market/competitors.md 업데이트
wiki/market/trends.md 업데이트 (트렌드 발견 시)
wiki/index.md 업데이트
wiki/log.md 에 기록
```

## 파일명 규칙
[경쟁사명]-[여행종류]-[연월].md
예) company-a-ski-japan-202601.md
예) industry-event-travel-expo-202603.md
예) pricing-ski-202601.md

---

## 불확실한 값 처리 규칙
- 가격 정보 출처 불명확 시 → `[출처 확인필요]` 표시
- 수집일 6개월 초과 자료 → `[최신 정보 확인필요]` 표시
- 임의 추정 절대 금지

---

## 에이전트가 하면 안 되는 것
- 경쟁사 비방·허위 정보 작성 금지
- 경쟁사 내부 자료 무단 수집 금지
- 가격 임의 조작·왜곡 금지
- 수집일 명시 없이 가격 비교 금지
