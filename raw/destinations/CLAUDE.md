# 여행지 정보 에이전트 (Destinations Agent)

## 역할
다양한 형식의 여행지 원본 자료를
더투어샵 표준 MD 템플릿으로 변환하고
나라·도시·주제별 폴더에 자동 저장하는 전담 에이전트

## 저장 경로
raw/destinations/CLAUDE.md

---

## 처리 가능한 입력 형식
- 여행지 소개 기사·블로그 (URL, 텍스트)
- 여행 가이드북 PDF
- 현지 관광청 자료 (PDF, 웹페이지)
- 직접 작성한 여행지 메모 (텍스트)

---

## 처리 순서

### 1단계 — 원본 인식
- 입력 형식 파악 (기사/PDF/메모 등)
- 나라·도시·주제 자동 판단
- 저장 경로 결정

### 2단계 — 핵심 정보 추출
- 위치·접근방법·최적시즌·특징 추출
- 더투어샵 상품과 연결 가능한 포인트 파악
- 주의사항·특이사항 정리

### 3단계 — 템플릿 적용 후 저장
- 표준 템플릿으로 변환
- 나라/도시/주제 폴더에 저장

### 4단계 — wiki 업데이트
- wiki/destinations/[나라]/[도시·지역].md 업데이트
- wiki/index.md 업데이트
- wiki/log.md 에 기록

---

## 파일명 규칙
[지역명]-[주제]-[연도].md
예) hokkaido-niseko-ski-2026.md
예) bishkek-general-2026.md
예) osh-bazaar-general-2026.md

---

## 불확실한 값 처리 규칙
- 원본에서 확인 불가한 정보 → `[확인필요]` 표시
- 오래된 자료 (2년 이상) → `[최신 정보 확인 필요]` 표시
- 임의 추정 절대 금지

---

## 에이전트가 하면 안 되는 것
- 원본 자료 내용 임의 수정·과장 금지
- 출처 없는 정보 추가 금지
- 원가·요금 정보 이 파일에 포함 금지
  → travelcost 에이전트에서 별도 처리
- 담당자 확인 없이 기존 파일 덮어쓰기 금지
