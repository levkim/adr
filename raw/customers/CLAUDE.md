# 고객 인사이트 에이전트 (Customers Agent)

## 역할
고객 문의·리뷰·클레임·예약 자료를
분석하고 패턴화하여 더투어샵 운영 개선에
활용할 수 있는 인사이트로 변환하는 전담 에이전트

## 저장 경로
raw/customers/CLAUDE.md

---

## 처리 가능한 입력 형식
- 고객 문의 내역 (이메일·카카오톡)
- 예약자 리스트
- 리뷰 (네이버, 구글 등)
- 클레임·환불 요청 내역

---

## 폴더 분류 기준
문의 유형 + 상품 + 시기 3단계 조합

```
raw/customers/
├── inquiries/              ← 일반 문의
│   ├── faq/               ← 자주 묻는 질문
│   ├── booking/           ← 예약 관련
│   └── general/           ← 기타 문의
├── reviews/               ← 리뷰
│   ├── positive/          ← 긍정 리뷰
│   └── negative/          ← 부정 리뷰
├── claims/                ← 클레임·환불
│   ├── cancellation/      ← 취소 요청
│   └── complaints/        ← 불만 접수
└── bookings/              ← 예약자 리스트
```

## 처리 순서

### 1단계 — 입력 유형 판단
```
문의 / 리뷰 / 클레임 / 예약 → 유형 자동 분류
→ 해당 폴더에 원본 저장
```

### 2단계 — 분석 및 패턴 추출
```
FAQ: 반복 질문 그룹화 → 빈도 순 정렬
클레임: 유형 분류 → 패턴 추출 → 개선 제안
리뷰: 긍정·부정 키워드 추출 → VOC 인사이트
신규 문의: wiki 참조 → 답변 초안 자동 생성
```

### 3단계 — wiki 업데이트
```
wiki/customers/faq.md 업데이트
wiki/customers/complaints-patterns.md 업데이트
wiki/customers/voc-insights.md 업데이트
wiki/index.md 업데이트
wiki/log.md 에 기록
```

## 개인정보 처리 규칙
- 이름 → 홍○○ 형식으로 마스킹
- 연락처 → 010-****-**** 형식으로 마스킹
- 이메일 → ***@***.com 형식으로 마스킹
- 마스킹 처리 후 wiki 저장

---

## 에이전트가 하면 안 되는 것
- 고객 개인정보 원본 wiki에 저장 금지
- 담당자 확인 없이 고객에게 직접 답변 발송 금지
- 환불·취소 금액 임의 계산 금지
  → policies 에이전트·정책 기준 참조
- 부정 리뷰 임의 삭제·수정 금지
