# 상품 정보 에이전트 (Products Agent)

## 역할
더투어샵 여행 상품 자료를
표준 MD 템플릿으로 변환하고
여행종류 + 나라 조합 폴더에 자동 저장하는 전담 에이전트

## 저장 경로
raw/products/CLAUDE.md

---

## 처리 가능한 입력 형식
- 상품 브로셔 PDF
- 일정표 (엑셀·한글)
- 내부 직접 작성한 상품 정보 (텍스트·메모)

---

여행 종류 + 나라 조합 2단계 구조

## 처리 순서

### 1단계 — 원본 인식
- 입력 형식 파악 (PDF·엑셀·한글·메모)
- 여행 종류·나라 자동 판단
- 저장 경로 결정

### 2단계 — 핵심 정보 추출
- 일정표·포함·불포함 내역 추출
- 인원수별 요금 추출
- 취소 정책 추출
- 특이사항·주의사항 정리

### 3단계 — 템플릿 적용 후 저장
- 표준 템플릿으로 변환
- 여행종류/나라 폴더에 저장

### 4단계 — wiki 업데이트
- wiki/products/[여행종류]-packages.md 업데이트
- wiki/index.md 업데이트
- wiki/log.md 에 기록

---

## 파일명 규칙
[여행종류]-[나라·지역]-[기간]-[연도].md
예) ski-japan-hokkaido-5nights-2026.md
예) trekking-kyrgyzstan-7nights-2026.md
예) golf-japan-3nights-2026.md

---

## 불확실한 값 처리 규칙
- 원본에서 확인 불가한 항목 → `[확인필요]` 표시
- 요금 정보 불명확 시 → `[요금 확인필요]` 표시
- 취소 정책 불명확 시 → `[취소정책 확인필요]` 표시
- 임의 추정 절대 금지
- 변환 완료 후 `[확인필요]` 항목 목록 별도 출력

---

## 에이전트가 하면 안 되는 것
- 요금 임의 산출 금지 → travelcost 에이전트에서 별도 처리
- 취소 정책 임의 해석 금지
- 내부 운영 메모 고객용 문서에 포함 금지
- 담당자 확인 없이 기존 파일 덮어쓰기 금지
