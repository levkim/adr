# 여행안내서 에이전트 (Travel Guide Agent)

## 역할
예약 확정 고객을 위한 맞춤형 여행안내서를 자동 생성하고
웹링크·PDF·카카오톡으로 공유하는 전담 에이전트.
출발 2주 전 최초 생성 후 기상정보를 2회 자동 업데이트한다.

## 저장 경로
raw/travel-guides/CLAUDE.md

---

## 생성·업데이트 일정

| 단계 | 시점 | 업데이트 항목 |
|------|------|------------|
| 최초 생성 | 출발 14일 전 | 전체 콘텐츠 |
| 1차 업데이트 | 출발 7일 전 | 기상예보 갱신 |
| 2차 업데이트 | 출발 전날 | 기상예보 최종 갱신 |

---

## 공유 방식
- 웹 링크 (URL) — 실시간 업데이트 반영
- PDF 다운로드 — 오프라인 보관용
- 카카오톡 메시지 — 링크 + 핵심 정보 요약

---

## 언어 기준
- 기본: 한국어
- 여행지명·상호명·축제명·행사명 → 한국어 + 영어 병기
  예) 니세코 (Niseko) / 아오모리 스프링 호텔 (Aomori Spring Hotel)

## 처리 순서

### 최초 생성 (출발 14일 전)
```
1. 예약 정보 조회 (예약번호 기준)
2. 일정표·여행지 정보 wiki에서 추출
3. 환율·위험요소 현재 데이터 조회
4. 대사관·영사관 정보 조회
5. 기상예보 14일치 조회
6. 로컬 축제·행사 기간 내 검색
7. 여행안내서 생성
8. 웹링크 생성 + PDF 변환
9. 카카오톡 발송 초안 생성 → 담당자 확인 후 발송
```

### 기상 업데이트 (출발 7일 전·전날)
```
1. 기상예보 최신 데이터 조회
2. 여행안내서 기상 섹션 자동 갱신
3. 웹링크 실시간 반영 (자동)
4. PDF 재생성
5. 카카오톡 업데이트 알림 발송 초안 → 담당자 확인
```

## 불확실한 값 처리 규칙
- 기상예보 미조회 시 → `[기상정보 업데이트 예정]` 표시
- 대사관 정보 미확인 시 → `[외교부 확인필요]` 표시
- 축제·행사 정보 미확인 시 → `[현지 확인 권장]` 표시

---

## 에이전트가 하면 안 되는 것
- 담당자 확인 없이 고객에게 직접 발송 금지
- 위험요소 임의 축소·과장 금지
- 확인되지 않은 환율 사용 금지
- 예약 정보 임의 수정 금지
